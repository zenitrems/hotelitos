# Hotelitos Search

**Client:** Vue, Vuetify, Axios

**Server:** Node, Express, AmadeusApi

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

## Environment Variables

**vue**

`VUE_APP_AXIOS_BASE_URL`

**node**

`NODE_ENV`

`PORT`

`AMADEUS_CLIENT_ID`

`AMADEUS_CLIENT_SECRET`
